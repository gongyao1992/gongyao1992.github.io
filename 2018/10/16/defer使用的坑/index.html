defer使用的坑

发表于 2018-10-16 | 分类于 开发语言

前言

defer是go的关键字之一，在开发使用defer的时候，经常会遇到坑

return的非原子操作

要了解defer，首先我们得明白go中return的非原子性。return语句并不是我们看着返回一个东西那么简单，return语句在go中是一个能够执行defer方法的信号，等defer方法执行完成之后再返回真正的数据

return 不是直接返回，而是一个执行defer的信号，执行完了之后再返回。 只要牢牢把握这个概念，defer就好理解了
