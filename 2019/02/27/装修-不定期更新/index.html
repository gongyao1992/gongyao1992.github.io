装修(不定期更新)

发表于 2019-02-27

前言

2019年6月底就要交房了，人生的第一套房子。交完房紧接着就是装修，在这里集众家之长把装修的思路以及注意事项梳理一下。

验房

携带东西

  • 直角尺
  • 卷尺5米
  • 纸
  • 笔（铅笔 + 白板笔）
  • 易贴便条
  • 试电笔（或其他即插型小电器）
  • 数码相机或DV
  • 小锤子
  • 矿泉水

检查项目

  1. 检查地面是否平整
  2. 卫生间下水管顶部有没有加装防火圈
  3. 地漏排水是否顺畅
  4. 门窗是否安装到位
  5. 水路验收做打压试验
  6. 检查楼上做是否漏水检查
